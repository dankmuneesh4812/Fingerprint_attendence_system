# Fingerprint_attendence_system
There are 3 code files uploaded in this repository.
The first is the I2C LCD driver which is the library for using DLCD with the Raspberry Pi 
The second file is the fingerprint recognition is used to initiate the fingerprint sensor to register the new fingerprint.
the last file is the fingerprint which is the main code that will send the e-mail as well as create a excel sheet for marking attendance.
